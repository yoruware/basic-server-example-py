# basic-server-example-tr-py
Socket metodu kullanılarak yapılmış başlangıç seviye bir server-client sistemi örneği.

How to Use

Bu kodlar, Python3 için uygundur bu yüzden sisteminizde Python3 yüklü olmalı. Dosyaları indirip terminal üzerinden cd komutuyla dosyaların bulunduğu dizine gidin, daha sonra python [dosyaadi].py ile çalıştırın. Windows kullanıyorsanız güvenlik uygulamaları ve firewall dosyaları çalıştırmanızı engelleyebilir.
